# AWS

Azure Web Services

**Cloud Coumputing** - доставка ІТ ресурсів за потребою за gay-as-you-go принципом.

**Multitenancy** - розподіл спільного заліза між віртуальними машинами.

**Vertial Scaling** - збільення потужності системи шляхом збільшення ресурсів інстансу.

**Horizontal Scaling** - Збільшення потужності системи шляхом збільшененя кількості серверів.

## Services

- **Computing**

  - **EC2** - (Amazon Elastic Compute Cloud) Віртуальні машини.

  - **EKS** - Managed Kubernetes Cluster, Кластер Кубернетес, яким керує AWS.

  - **ELB** - (Elastic Load Balancing) балансувальник навантаження який керуєится AWS.

  - **ECS** - Container Service. Сервіс який дозволяє деплойнути лише один контейнер.

  - **Lambda** - Дозволяє створювати Serverless застосунки.Оплата стягується за кількість запусків застосунку.

- **Storage**

  - **S3** - Дешевий сервіс для збереження даних.
  - **RDS** - PaaS сервіс для розгортання баз даних.

### EC2

Amazon EC2 (Elastic Compute Cloud) — це вебсервіс, який надає безпечні віртуальні сервери (інстанси) у хмарі з можливістю масштабування.

EC2 потребує `AMI` (Amazon Machine Image) для роботи. АМI можна отримати в 3 способи: завантажти власну, вибрати готову із магазину, купити платну із магазину.

EC2 ділиться на `types`, кожен із яких надає різні конфігурації інстансу (RAM, CPU, GPU, Storage, Networking)

Типи:

- **General Purpose** - Загальні для більшості завдань.
- **Compute Optimised** - Для складних обчислень.
- **Memory Optimized** - Для залежних від пам'яті задач.
- **Accelerated Computing** - Для обчислень, які вимагають участь графічного процесора.
- **Storage Optimized** - Для потреб у великому обсязі хранилища.

Billing:

- **Reserved Instances** - це підписка на хмарні ресурси, яка дозволяє отримати значні знижки в обмін на зобов’язання використовувати певний обсяг ресурсів протягом одного до трьох років.

- **On Denand Instances** - Оплата за умовою скільки використав стільки і заплатив.

- **Saving Instances** - договір про використання інстансу сталий час на місяці протягом n місяців за сталою ціною.

- **Spot Instances** - це невикористані хмарні ресурси, які продаються за значно зниженими цінами, але можуть бути відключені провайдером, коли ці ресурси займаються більш приорітетними ресурками, наприклад Reserved Instances, On Demand

- **Dedicated Hosts** - Оренда цілого фізичного сервера.

- **Dedicated Instances** - Послуга, яка гарантує, що сервер, на якому запущений шан ністанс не міститиме ресурси інших клієнтів.

Scaling:

- **Scalability** - здатність EC2 збільшувати як ресурси на одиницю, так і кількість одиниць
- **Elasticity** - здатність інстансу автоматично скалювати.
